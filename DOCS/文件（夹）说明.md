# 生成项目目录

``` js

$ npm install mddir -g
$ mddir "./"

```

# 目录结构说明

``` js

|-- 项目根目录
    |-- .gitignore                  git提交忽略文件
    |-- config-overrides.ts         配置重新文件，如antd-mobile、swiper、less模块化等        
    |-- package-lock.json           项目依赖包
    |-- package.json                项目基础信息及依赖包信息                
    |-- README.md                   使用教程、文档
    |-- tsconfig.json               typescript配置文件
    |-- yarn.lock                   
    |-- DOCS                        项目文档说明
    |   |-- 文件（夹）说明.md
    |-- public                      公共资源文件，不会被打包
    |   |-- favicon.ico
    |   |-- index.html              单页应用首页html
    |   |-- logo192.png
    |   |-- logo512.png
    |   |-- manifest.json           移动App的配置文件
    |   |-- robots.txt              爬虫协议、爬虫规则、机器人协议 
    |-- src                         react源文件
        |-- App.css
        |-- App.test.tsx
        |-- App.tsx
        |-- index.css
        |-- index.tsx
        |-- logo.svg
        |-- react-app-env.d.ts
        |-- serviceWorker.ts
        |-- setupTests.ts
        |-- setupProxy.ts           react本地api代理配置文件                   



```
